# 🏰 PyQuest — Dungeon Crawler do nauki Pythona

Interaktywna gra webowa w klimacie dungeon crawlera, która uczy Pythona od podstaw do poziomu zaawansowanego.

## Funkcje

- 🗡️ **5 rozdziałów** (lochów) — od Hello World po OOP
- ⚔️ **4 typy pytań** — Quiz, Uzupełnij luki, Ułóż kod, Napisz kod
- 🏆 **System XP i poziomów** — zdobywaj doświadczenie za poprawne odpowiedzi
- 📜 **Karta postaci** — śledź swój postęp i tytuły (Adept → Mistrz Zaklęć)
- 🏅 **Tablica chwały** — ranking wszystkich graczy
- 🎨 **Klimat dungeon** — ciemna paleta, złote akcenty, czcionka Cinzel

## Technologie

- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** Express.js + SQLite (Drizzle ORM)
- **Język:** TypeScript

## Uruchomienie lokalne

```bash
npm install
npm run dev
```

Aplikacja uruchamia się na `http://localhost:5000`.

## Build produkcyjny

```bash
npm run build
NODE_ENV=production node dist/index.cjs
```

## Zawartość kursu

| Rozdział | Poziom | Lekcje |

| Podstawy Pythona | Nowicjusz | Hello World, Zmienne, Typy danych |
